# 虚拟歌手中文曲排行榜 - 特定视频抓取&计分工具
该工具用于抓取B站指定BV号视频的播放/互动数据，并基于预设规则计算视频的周刊得分、生成排行榜，适配虚拟歌手中文曲排行榜的计分逻辑。

## 功能特点
1. **精准抓取**：指定BV号列表，批量抓取视频的播放、弹幕、评论、收藏、投币、点赞等核心数据；
2. **增量计算**：对比两期抓取的CSV数据，自动计算周期内各项数据的增量；
3. **计分排名**：按照虚拟歌手中文曲排行榜的计分规则，计算视频最终得分并生成降序排行榜；
4. **容错处理**：包含BV号格式校验、请求限流重试、异常捕获等机制，保证抓取稳定性；
5. **编码兼容**：生成的CSV文件采用`utf-8-sig`编码，支持中文正常显示。

## 环境准备

确保电脑已安装Python（推荐3.8及以上版本），下载地址：https://www.python.org/downloads/

### 2. 安装依赖库
打开终端/命令提示符，执行以下命令安装所需依赖：
```bash
pip install requests pandas
```

## 使用步骤
### 步骤1：运行工具
将工具脚本保存到本地，双击运行（或在终端执行），程序会显示主菜单：
```
==================================================
虚拟歌手中文曲排行榜 特定视频抓取&计分工具
==================================================
1. 抓取指定BV号的视频数据并保存
2. 计算增量、得分并生成排行榜
3. 退出
==================================================
请输入操作序号：
```

### 步骤2：抓取视频数据（选1）
选择`1`后，支持两种BV号输入模式：
#### 模式1：使用预设BV号列表
直接修改脚本中`DEFAULT_BVID_LIST`变量，添加需要抓取的BV号（格式：`"BVxxxxxx"`，英文逗号分隔）：
```python
DEFAULT_BVID_LIST = [
    "BV1LXwyzkEqo",
    "BVxxxxxx",  # 新增BV号
]
```
运行后选择模式1，工具会自动加载列表并抓取数据，生成以「当日日期」命名的CSV文件（如`20260324_specified_video_data.csv`）。

#### 模式2：手动输入BV号
选择模式2后，按提示输入BV号（多个BV号用**空格**分隔），例如：
```
请输入要抓取的BV号，多个BV号用空格分隔：BV1LXwyzkEqo BVxxxxxx
```
工具会抓取输入的BV号数据并生成CSV文件。

### 步骤3：计算得分并生成排行榜（选2）
选择`2`后，按提示输入**上期数据CSV文件**和**本期数据CSV文件**（需为步骤2生成的文件）：
```
请输入上期数据CSV文件名（如20260317_specified_video_data.csv）：20260317_specified_video_data.csv
请输入本期数据CSV文件名（如20260324_specified_video_data.csv）：20260324_specified_video_data.csv
```
工具会：
1. 匹配两期数据中相同的BV号；
2. 计算周期内各项数据的增量；
3. 按计分规则计算最终得分；
4. 按得分降序生成排行榜，保存为`specified_rank_result.csv`；
5. 终端预览排名（包含排名、BV号、标题、UP主、最终得分）。

## 核心配置说明
脚本中可调整的配置项（无需修改即可使用）：
| 配置项          | 说明                                                                 |
|-----------------|----------------------------------------------------------------------|
| `REQUEST_INTERVAL` | 请求间隔（秒），默认2秒，避免触发B站接口限流                        |
| `RETRY_TIMES`      | 失败重试次数，默认3次，网络波动时自动重试                            |
| `HEADERS`          | 请求头，模拟浏览器行为，无需修改                                    |

## 计分规则说明
最终得分由以下维度加权计算（严格匹配虚拟歌手中文曲排行榜规则）：
1. **播放得分**：基础播放分 × 修正系数D；
2. **互动得分**：（评论+弹幕）× 修正系数A × 15；
3. **收藏得分**：收藏数 × 修正系数B；
4. **投币得分**：投币数 × 修正系数C；
5. **点赞得分**：点赞数（若点赞>投币×2，取投币×2）；
最终得分 = 播放得分 + 互动得分 + 收藏得分 + 投币得分 + 点赞得分。

## 注意事项
1. BV号格式需为`BV+10位字符`（共12位），格式错误会被自动跳过；
2. 抓取频率请勿过高（默认间隔2秒），避免被B站接口限流；
3. 若提示「触发限流」，工具会自动暂停30秒重试，无需手动干预；
4. 生成的CSV文件请不要修改列名，否则会导致增量计算失败；
5. 两期数据需包含相同的BV号，否则无法计算增量。

## 常见问题
### Q1：抓取失败提示「视频不存在/已被删除」？
A：确认BV号是否正确，或视频是否已被UP主删除/设置为私密。

### Q2：读取CSV文件提示编码错误？
A：确保CSV文件为工具生成的`utf-8-sig`编码，避免手动修改后保存为其他编码（如GBK）。

### Q3：排行榜生成后部分视频缺失？
A：缺失的视频为两期数据中未匹配到的BV号（仅计算两期都存在的视频）。
