# 虚拟歌手中文曲排行榜 计分排行工具
基于 Streamlit 开发的B站虚拟歌手中文曲数据抓取与排行榜计分工具，支持批量抓取视频数据、计算周期数据增量并生成计分排行榜。

## 功能说明
1. **视频数据抓取**：支持预设BV号列表或手动输入多个BV号，批量抓取B站视频的播放、弹幕、评论、收藏、投币、点赞等核心数据，支持CSV格式导出。
2. **排行榜生成**：上传两期（上期/本期）抓取的CSV数据，自动计算周期内各视频的数据增量，按照自定义计分规则计算最终得分并生成排行榜，支持完整数据CSV导出。

## 环境准备
### 1. 安装Python环境
确保已安装Python 3.8及以上版本。

### 2. 安装依赖包
执行以下命令安装所需依赖：
```bash
pip install streamlit requests pandas
```

## 使用步骤
### 1. 运行工具
在终端/命令行中进入工具所在目录，执行以下命令启动Web界面：
```bash
streamlit run 虚拟歌手中文曲排行榜 计分排行工具.py
```
启动成功后，会自动打开浏览器（默认地址：`http://localhost:8501`）。

### 2. 抓取视频数据（📥 抓取视频数据标签页）
#### 方式1：使用预设BV号列表
- 工具内置默认BV号列表（可在代码`DEFAULT_BVID_LIST`中修改），选择「使用预设列表」后，直接点击「开始抓取」。

#### 方式2：手动输入BV号
- 选择「手动输入」，在输入框中填写需要抓取的BV号（多个BV号用空格分隔，例如：`BV1LXwyzkEqo BVxxxxxx`）。
- 点击「开始抓取」，等待抓取完成。

#### 结果导出
- 抓取成功后，页面会展示数据表格，并提供「📤 下载CSV数据」按钮，点击可下载当期抓取的完整数据（文件名格式：`YYYYMMDD_specified_video_data.csv`）。

### 3. 生成排行榜（🏆 生成排行榜标签页）
- 点击「上传上期数据CSV」，选择历史抓取的CSV文件；
- 点击「上传本期数据CSV」，选择最新抓取的CSV文件；
- 点击「生成排行榜」，工具会自动匹配两期数据中相同的BV号，计算数据增量和最终得分；
- 生成成功后，页面会预览核心排行数据（排名、BV号、标题、UP主、最终得分），并提供「📤 下载完整排行榜」按钮，点击可下载包含所有计分维度的完整排行榜数据。

## 核心规则说明
### 数据抓取规则
- 抓取间隔：默认每次请求间隔2秒（`REQUEST_INTERVAL`），避免触发B站接口限制；
- 重试机制：接口请求失败时默认重试3次（`RETRY_TIMES`）；
- 请求头：模拟浏览器请求（`HEADERS`），提升接口请求成功率。

### 计分规则
工具通过`calculate_weekly_score`函数实现自定义计分逻辑，核心维度包括：
- 播放得分、互动得分（弹幕+评论）、收藏得分、投币得分、点赞得分；
- 包含多维度修正系数（修正A/B/C/D），可根据需求在函数内调整权重。

## 注意事项
1. BV号格式：必须为B站标准BV号（以BV开头，后接10位字符），多个BV号需用空格分隔；
2. 编码问题：导出的CSV文件采用`utf-8-sig`编码，确保中文内容正常显示；
3. 数据匹配：生成排行榜时，两期数据需包含相同的BV号，否则会提示「无匹配的BV号」；
4. 接口限制：请勿频繁抓取大量数据，避免触发B站接口限流/封禁；
5. 自定义调整：可修改代码中`DEFAULT_BVID_LIST`（预设BV号）、`REQUEST_INTERVAL`（请求间隔）、计分函数等参数适配实际需求。

## 常见问题
- **抓取数据为空**：检查BV号是否有效、网络是否正常、B站接口是否临时调整；
- **CSV文件乱码**：确保用Excel/WPS以「utf-8编码」打开，或使用记事本打开后另存为utf-8格式；
- **排行榜计算报错**：检查两期CSV文件的列名是否一致，确保数据格式未被手动修改。
