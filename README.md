# 虚拟歌手中文曲排行榜 - 特定视频抓取&计分工具
该工具用于抓取B站指定BV号视频的播放/互动数据，并基于预设规则计算视频的周刊得分、生成排行榜，适配虚拟歌手中文曲排行榜的计分逻辑。

## 功能特点
1. **多方式输入BV号**：支持预设列表、手动输入、文件导入（TXT/CSV/Excel）三种方式，自动校验并去重BV号
2. **稳定的数据抓取**：带重试机制和请求间隔，避免请求过于频繁导致失败
3. **增量计算与排行榜**：对比两期数据，计算各维度增量并按自定义规则计分，生成排序后的排行榜
4. **友好的输出**：进度条展示抓取进度，清晰的日志提示，结果保存为UTF-8编码的CSV文件（兼容Excel打开）

## 环境要求
- Python 3.7+
- 依赖库：
  - requests：发送HTTP请求
  - pandas：数据处理与文件读写
  - tqdm：进度条展示
  - openpyxl（可选）：处理Excel文件（.xlsx/.xls）

## 安装依赖
执行以下命令安装所需依赖：
```bash
pip install requests pandas tqdm openpyxl
```

## 使用说明

### 1. 运行脚本
```bash
python demo.py
```
运行后会显示主菜单，选择对应功能序号即可：
```
==================================================
虚拟歌手中文曲排行榜 特定视频抓取&计分工具
==================================================
1. 抓取指定BV号的视频数据并保存
2. 计算增量、得分并生成排行榜
3. 退出
==================================================
请输入操作序号：
```

### 2. 功能1：抓取视频数据
选择序号`1`后，进入BV号输入模式选择：
```
请选择BV号输入模式：
1. 使用脚本预设的BV号列表
2. 手动输入BV号
3. 从文件批量导入BV号
请输入模式序号：
```

#### 模式1：使用预设BV号
脚本中`DEFAULT_BVID_LIST`变量已预设BV号，选择后自动加载并校验有效性。

#### 模式2：手动输入BV号
输入多个BV号，用空格分隔，示例：
```
请输入要抓取的BV号，多个BV号用空格分隔：BV1LXwyzkEqo BV12345678901
```
脚本会自动校验格式，仅保留有效BV号。

#### 模式3：从文件导入BV号
支持以下文件格式：
- **TXT**：每行/空格/逗号分隔BV号
- **CSV**：任意列中包含BV号，自动遍历所有列提取
- **Excel（.xlsx/.xls）**：任意列中包含BV号，自动遍历所有列提取

输入文件路径（相对/绝对路径），示例：
```
请输入BV号文件路径（如 bv_list.txt）：./bv_list.csv
```

#### 抓取结果
- 成功抓取的视频数据会保存为CSV文件，命名规则：`YYYYMMDD_specified_video_data.csv`
- 日志会显示抓取成功/失败数量，以及每个视频的抓取状态

### 3. 功能2：计算增量与生成排行榜
选择序号`2`后，需要输入上期和本期的抓取数据CSV文件名称（含路径，如无路径则为当前目录）：
```
请输入上期数据CSV文件名：20240501_specified_video_data.csv
请输入本期数据CSV文件名：20240502_specified_video_data.csv
```

#### 计算规则
- 增量计算：本期数值 - 上期数值（最小值为0）
- 计分规则（可自定义）：
  | 维度       | 权重/计算方式                |
  |------------|-----------------------------|
  | 播放得分   | 播放增量 × D（默认1）        |
  | 互动得分   | (评论增量+弹幕增量) × A ×15（默认A=1） |
  | 收藏得分   | 收藏增量 × B（默认1）        |
  | 投币得分   | 投币增量 × C（默认1）        |
  | 点赞得分   | 点赞有效增量（≤ 投币增量×2） |
  | 最终得分   | 以上所有得分之和             |

#### 输出结果
- 排行榜文件：`specified_rank_result.csv`
- 控制台会预览TOP10视频的排名、BV号、标题、UP主、最终得分
- 完整排行榜包含：排名、BV号、本期标题、本期UP主、最终得分、各维度增量

### 4. 功能3：退出
选择序号`3`，程序正常退出。

## 注意事项
1. **BV号格式**：仅支持标准BV号（长度12位，以BV开头），脚本会自动过滤无效格式
2. **请求限制**：B站API有请求频率限制，请勿修改`REQUEST_INTERVAL`为过小值，否则可能导致抓取失败
3. **文件编码**：
   - TXT文件支持UTF-8/GBK编码自动识别
   - CSV/Excel文件建议使用UTF-8编码
4. **数据匹配**：计算排行榜时，仅匹配两期数据中相同的BV号
5. **异常处理**：网络异常、视频下架/删除会导致抓取失败，脚本会记录失败信息并继续处理下一个BV号

## 常见问题
1. **Excel文件读取失败**：确保安装了openpyxl（`pip install openpyxl`）
2. **抓取失败提示“视频不存在”**：BV号错误或视频已下架/私密
3. **CSV文件打开乱码**：使用Excel打开时选择“UTF-8编码”，或用记事本打开后另存为ANSI编码
4. **请求超时/重试多次失败**：检查网络，或增大`REQUEST_INTERVAL`/`RETRY_TIMES`
